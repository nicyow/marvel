# Web Application 

## Requirements

 - [Node v16.8.0](https://nodejs.org/en/download/current/)

## Getting Started

#### Clone the repo and make it yours:

```bash
git clone https://github.com/nicyow/marvel.git
cd marvel-app
rm -rf .git
```

#### Install dependencies:

```bash
npm i
```

## Running with nodemon

```bash
npm run dev
```

## Running without nodemon

```bash
npm run start
```

## Test

```bash
npm test
```
